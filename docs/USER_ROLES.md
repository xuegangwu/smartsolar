# SmartSolar 用户角色定义

## 角色总览

| 角色 | 职责 | 核心操作 |
|------|------|---------|
| 👨‍💼 **管理员 Admin** | 系统运维 | 全部功能 |
| 👨‍💻 **值班员 Operator** | 监控调度 | 告警确认、工单派发、巡检执行 |
| 🔧 **技术员 Technician** | 现场作业 | 接收工单、反馈进度、消耗备件 |
| 📊 **主管 Manager** | 数据分析 | 查看KPI、导出报告、审批流程 |

---

## 1. 管理员 (Admin)

**职责：** 系统配置 + 数据管理

- 电站 CRUD（增删改查）
- 设备台账管理
- 备件仓库管理（添加/删除备件类型）
- 用户账号管理（增删改查）
- 巡检计划模板配置
- 系统参数配置（阈值、告警规则）
- EMS 同步接口配置

**当前实现：** `admin` 账号 = 管理员权限

---

## 2. 值班员 (Operator)

**职责：** 7×24 实时监控 + 工单调度

- 实时监控 Dashboard（告警统计、工单状态）
- 告警确认 / 批量确认
- 告警转工单（一键生成）
- 工单派发（指派给技术员）
- 巡检计划发起（周期任务生成执行记录）
- 备件申请 / 领用
- EMS 模拟器测试

**当前实现：** 与 Admin 共用同一套 UI，权限未分离

---

## 3. 技术员 (Technician)

**职责：** 现场执行 + 反馈

- 接收派发工单（工单列表 → 接受）
- 更新工单状态（处理中 → 暂停 → 完成）
- 填写工单处理记录（故障描述、解决方案）
- 消耗备件（从仓库扣减库存）
- 提交巡检执行记录（结果、照片、备注）

**当前实现：** 与 Admin 共用同一套 UI，权限未分离

---

## 4. 主管 (Manager)

**职责：** 运营分析 + 决策支持

- 查看 KPI 面板（可用率/PR/MTTR/成本）
- 数据导出（CSV）：工单/告警/巡检/备件
- 电站拓扑图 / 数字孪生（总览）
- 历史数据查询（时间范围筛选）
- 报表生成（月报/年报）

**当前实现：** 与 Admin 共用同一套 UI，权限未分离

## 当前状态

目前平台为**单租户内部版**，所有用户共用 `admin` 账号，权限未做分离。

如需多角色支持，需要扩展：
1. 用户管理 API（CRUD + 角色分配）
2. 登录认证（JWT / Session）
3. 路由守卫（基于角色的页面访问控制）
4. API 权限中间件
