## 三、技术债 & 隐患

### 1. 设备分类路由 Bug
- `/api/categories` 返回 404（路由只注册了 `/api/stations/:stationId/categories`）
- 设备新建/编辑依赖此接口，导致 Equipment 模块无法正常使用

### 2. PM2 重启后 MongoDB 记忆
- PM2 重启后有时连到 memory MongoDB（mongodb-memory-server）而非真实 MongoDB
- 需要固定 `.env` 路径或改用 `pm2-runtime`

### 3. 前端构建物同步
- 每次修改需手动 rsync
- GitHub Actions Secrets 仍未配置，自动化部署未打通

### 4. Auth 路由中动态 import Personnel
- `authRoutes.ts` 里 `import { Personnel } from '../models/index.js'` 可能导致循环依赖
- 当前正常但存在隐患
